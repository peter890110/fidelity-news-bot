# AI 財經新聞分析助理 (AI Financial News Analyst Assistant)

一個本地運行的智慧財經新聞聚合與解碼助理。每天早晨或盤前、盤後，自動抓取《經濟日報》、《工商時報》及國際/美股重大訊息，自動篩選 5-7 則最具市場影響力的故事，並依據特定格式進行深度分析、標示關鍵數據、交易信號（看多/看空/中立），並提供一鍵複製報告功能。

---

## 特色功能

- **來源保證**：至少包含一則《經濟日報》及一則《工商時報》的最新報導。
- **高時效性**：預設篩選 12-24 小時內之最新網頁新聞。
- **優雅設計**：採用 Obsedian 深色主題、Glassmorphism 玻璃擬物設計，具備動態進度加載、流暢卡片懸停與亮眼信號標記。
- **雙向 Key 設定**：可於 `.env` 中設定 `GEMINI_API_KEY`，亦可直接於網頁介面的「設定」中填寫（安全地儲存於您本地瀏覽器的 `LocalStorage`）。
- **一鍵報告複製**：符合您指定的輸出格式，方便立即轉貼或分享。

## 快速開始

### 1. 啟動伺服器

專案已經預先為您建立了 Python 虛擬環境並安裝好所有依賴。您只需在專案目錄下執行以下指令啟動 Flask 伺服器：

```bash
# 確保您在此專案目錄下下
./venv/bin/python3 app.py
```

伺服器將在本地運作： `http://127.0.0.1:8010`

### 2. 打開瀏覽器

在瀏覽器中開啟以下網址：
[http://127.0.0.1:8010](http://127.0.0.1:8010)

### 3. 設定 Gemini API 金鑰

1. 點擊網頁右上角的 **齒輪按鈕** ⚙️。
2. 填入您的 Gemini API Key（`AIzaSy...` 開頭）。
3. 選擇您偏好的分析引擎（預設 `Gemini 1.5 Flash`，可選 `Gemini 2.5 Flash` 或 `Gemini 1.5 Pro`）。
4. 點擊 **儲存金鑰**。

### 4. 獲取解析

點擊主畫面的 **獲取最新財經解析** 即可。分析完成後，點擊右上角 **複製完整分析報告** 即可獲得符合以下格式的中文文字報告：

```text
[CATEGORY]
HEADLINE: 一句話新聞摘要
IMPACT: 受影響的對象及如何受影響
KEY NUMBER: 最重要的數據或指標
SIGNAL: Bullish / Bearish / Neutral — 一句話原因
```
